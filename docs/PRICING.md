# BIG Oakland Virtual Mail - Pricing Guide

*Last Updated: December 2024*

This document summarizes the pricing structure for BIG Oakland's virtual mail services. All prices are admin-configurable through the Mail Center Portal.

## Mail Forwarding

### Handling Fee
A flat **$3.00 handling fee** is added to all forwarded mail, plus the actual shipping cost (with margin).

### Consolidation
Combine multiple items into one shipment: **$5.00** (saves on shipping)

### Shipping Carriers & Margins

## Example Calculations

### Scan Request (15-page document)
```
Base scan (10 pages):     $2.50
Additional (5 pages):     $1.50 (5 × $0.30)
─────────────────────────────────
Total:                    $4.00
```

### Mail Forwarding (Priority Mail, $8.50 postage)
```
Carrier postage:          $8.50
Margin (20%):             $1.70 ($8.50 × 0.20)
Handling fee:             $1.00
Forwarding handling:      $3.00
─────────────────────────────────
Total:                    $14.20
```

### Storage (45 days, 3 items)
```
First 30 days:            FREE
Days 31-45 (15 days):     $4.50 (3 items × 15 days × $0.10)
─────────────────────────────────
Total:                    $4.50
```
